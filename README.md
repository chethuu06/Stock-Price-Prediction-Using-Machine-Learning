# Stock Price Prediction Using Machine Learning

This project predicts future stock price trends using machine learning techniques. 
Historical stock market data is analyzed and regression algorithms are applied to 
identify patterns and generate predictions.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Features
- Data preprocessing and analysis
- Model training using regression algorithm
- Price trend visualization
- Future price prediction

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run:
   python stock_prediction.py
